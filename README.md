# MySQL Database Optimization using Indexing

## Objective

This project demonstrates how to optimize a MySQL database using indexing and query optimization techniques.

## Tools Used

- MySQL 8.x
- MySQL Workbench

## Features

- Create Database
- Create Student Table
- Insert Sample Data
- Analyze Queries using EXPLAIN
- Create Indexes
- Optimize Queries
- Monitor Performance
- Optimize Table
- Analyze Table

## SQL Concepts Used

- CREATE DATABASE
- CREATE TABLE
- INSERT
- SELECT
- EXPLAIN
- CREATE INDEX
- SHOW INDEX
- GROUP BY
- ORDER BY
- AVG()
- COUNT()
- MAX()
- MIN()
- OPTIMIZE TABLE
- ANALYZE TABLE
